## Important stuff (read if you want to know how to draw etc.)

### What you need to install:

* Every lib from `requirements.txt` just open powershell/CMD in your main project folder and type `pip install -r requirements.txt`
* You will need to install Ghostscript

### If running code always check working directory if it's not `path/to/irtTurtle.py` change it to it because saving may not work in other case

### Basic movement:

* **'W'** → forward
* **'S'** → backward
* **'A'** → turn 15° left
* **'D'** → turn 15° right

### Other basics that you will 100% need:

* **'x'** → quits app
* **'↑'** → stop drawing (e.g. to move pen somewhere else without making line)
* **'↓'** → start drawing (need to press after starting)
* **'f7'** → hold/unhold key work (e.g. when you need to type message)
* **'PgUp'** → faster mode (default)
* **'PgDn'** → slower mode
* **'←'** → thinner line
* **'→'** → thicker line
* **'k'** → change pen color
* **'ctrl+s'** → save your work (saves as `.ps` in `psFiles/` and as `.png` in `imagesPNG/`)
* **'n'** → clean canva so you have fresh start without quiting

### Other useful stuff:

* **'insert'** → codes (there will be some easter eggs someday maybe) for now there's only "Help!" for help (currently opening page is github because I still don't have my own)
* **'e'** → lets you customize speed (0.0 is default 1.0 moves 1px/s)
* **'h'** → changes pen shape
* **'k'** → is used to change pen color
* **'0'** → instantly moves pen to starting position (MAKES LINE! so be cautious)
* **'g'** → instantly moves pen to position that you want (MAKES LINE TOO!)
* **'p'** → let you change line thickness
* **'o'** → other way to set pen and fill colors
* **'b'** → changes background color
* **'t'** → turn options (custom angle), then press second key:<br>
    a) **'l'** (small L) to turn left<br>
    b) **'r'** to turn right

### Geometrics:

* **'c'** → makes circles
* **'l'** (small L) → makes regular polygons

### Fill functions, **'f'** then:

* **'b'** → begin fill 
* **'e'** → end fill 
* **'k'** → change fill color
* **'esc'** → goes back to normal without any changes

### Color list:

* *0* = white
* *1* = black
* *2* = red
* *3* = orange
* *4* = yellow
* *5* = green
* *6* = blue
* *7* = violet
* *8* = pink
* *9* = grey
